# JOBX

A modern job board platform built with Next.js, TypeScript, and Supabase.

## Features

- Job seeker and company dashboards
- Authentication (login/signup)
- Admin panel
- Responsive UI with Shadcn components

## Getting Started

The main application is in the `joblink/` directory.

1. Clone the repo
2. `cd joblink`
3. `npm install`
4. Set up Supabase environment variables
5. `npm run dev`

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Shadcn/UI
